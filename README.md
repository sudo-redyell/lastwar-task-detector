"# lastwar-task-detector" 

1. Run the .exe file.
2. Select 'Last War - Survival Game' from the dropdown list.
3. Adjust the zoom level on the 'Last War' game window's world map.
Set it so that 3 golden chests are visible.
4. Click the 'Start' button.
5. The 'Last War' world map will move approximately every second.
6. When a chest is detected, a beep will sound, and the mouse cursor will move to it.
